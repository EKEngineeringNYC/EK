# My Personal Page

A simple personal website built with HTML and CSS — perfect for beginners.

## How to view it

1. Open the folder `my-personal-page` in File Explorer.
2. Double-click `index.html` — it will open in your default browser.

Or in Cursor: right-click `index.html` → **Open with Live Preview** (if you have that extension).

## Customize it

Edit `index.html` to change:

- Your name in the `<h1>`
- The about text, skills, and projects
- Your email and social links

Edit `styles.css` to change colors — look for `#4f46e5` and `#7c3aed` (purple/indigo theme).

## Files

| File         | Purpose                          |
|--------------|----------------------------------|
| `index.html` | Page structure and content       |
| `styles.css` | Colors, spacing, and layout      |

No install or build step required — just HTML and CSS.
